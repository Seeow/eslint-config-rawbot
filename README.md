# eslint-config-rawbot

Rawbot EsLint common configuration (i.e. JavaScript styleguide).

## Usage

Install the package first.

```
npm install --save-dev eslint-config-rawbot
```

Set your EsLint configuration (.eslintrc.json) to use it:

```
"extends": "rawbot"
```
